Vanilla JS ist einfach reines Java Script ohne zusätzliche Frameworks. 
Ein Vorteil nur Java Skript zu verwenden ist einmal die Geschwindigkeit. Das wirkt sich sowohl beim laden der Webseite aus, da nicht die Dateien der verwendeten Frameworks geladen werden müssen (welche eine erhebliche Größe haben können). Aber auch zur Laufzeit erreicht einfaches Java Script wesentlich mehr Durchsatz. Die Lernkurve ist flacher bei der Verwendung von Villa JS man muss nur Java script lernen und nicht für jedes Framework seinen speziellen Syntax. Was die Einarbeitung in Projekte einfacher macht. 
Ein Nachteil ist das man alle Funktionen die nab benötigt selber programmieren muss, die Frameworks haben oft verwendete Funktion schon implementiert. Viele Frameworks bieten Syntaxsugar um das Programmierung zu beschleunigen.
